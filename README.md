# Responsive-Flower-Website-page-with-HTML-CSS

A simple and fully **responsive flower shop website** built using **HTML and CSS**.  
The layout adjusts smoothly for mobile, tablet, and desktop screens.

Link: https://github.com/ProgrammingHero1/B12A02-Responsive-Flowers
---

##  Features
- Responsive design using media queries  
- Navigation bar and banner section  
- Flower/product cards  
- Footer section  

---

##  Technologies
- HTML5  
- CSS3
